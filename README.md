# Proyecto_Juego
Este repositorio contiene los archivos relacionados a la creación y funcionamiento del videojuego para la clase de Construcción de Software y Toma de Decisiones

Las instrucciones del juego de encuentran en el archivo README de la carpeta Videojuego.

La carpeta BackEnd contiene los archivos relacionados al servidor y api´s del juego.

La carpeta DataBase contiene los archivos de la creación de la base de datos y la carga de los datos.

La carpeta Documentos contiene los documentos relacionados al diseño y planeación del juego.

La carpeta Presentacion contiene el archivo de la presnetación final del videojuego.

La carpeta sprites contiene sprites varios que se utilizan en ciertos momentos del juego.

La carpeta Tareas contiene las tareas que se reaizaron en equipo.

La carpeta Videojuego contiene los archivos relacionados al videojuego en sí, y las escenas.

La carpeta Web contiene los archivos de las distintas páginas que conforman la web del videojuego.
